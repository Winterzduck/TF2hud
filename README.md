# "Ace" hud
Re-edit of Hudas Iscariote (by Mangachu) for Team Fortress 2.

Edit the hudlayout.res to your liking. The Fog crosshair "2" is enabled by default, so make sure to use crosshair 0 in TF2.

Included crosshairs:

* Rainbow_Konr wings
* Quartz Crosshairs
* Fog Crosshairs
* Garm3n crosshairs
